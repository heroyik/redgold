# 중국어 단어장 앱 요구사항 정의서 (v2.0)

## 1. 프로젝트 개요
* **목표:** HSK Standard Course 4A/4B 교재 기반의 모바일 우선(Mobile-First) 복습 앱 개발.
* **학습 소스:** 
    1. HSK Standard Course 4A Textbook
    2. HSK Standard Course 4B Textbook
* **핵심 차별점:** 교재의 문어체 내용을 분석하여 실생활 **구어체(Colloquial)** 표현으로 확장 학습 제공.

## 2. 핵심 워크플로우 (Core Workflow)
1. **이미지 캡처 및 분석:** 사용자가 교재의 챕터별 본문/문법 페이지 이미지를 제공.
2. **콘텐츠 추출 (AI 분석):** AI가 이미지 내 한자, 병음, 의미, 문법 패턴을 추출.
3. **구어체 확장:** 추출된 문어체 표현과 대응되는 실생활 구어체 표현을 자동으로 생성/추가.
4. **모바일 복습:** 추출된 데이터를 기반으로 최적화된 모바일 UI에서 복습 진행.

## 3. 기능적 요구사항
* **이미지 분석 엔진:** 교재 이미지를 OCR 및 문맥 분석하여 학습 데이터셋으로 변환.
* **구어체 큐레이션:** HSK 4급 단어를 활용한 원어민 수준의 구어체 예문 및 뉘앙스 설명 제공.
* **멀티미디어 연동:** 교재 부록 및 MP3 자료(가능한 경우)를 활용한 리스닝 보조 기능.
* **챕터별 관리:** 4A, 4B 교재의 과(Lesson)별 진도 관리.

## 4. 기술적 요구사항
* **프레임워크:** Vanilla TypeScript + Web Components (GitHub Pages 배포).
* **백엔드/AI:** Google Cloud Vertex AI (이미지 분석 및 구어체 생성), Firebase (데이터 저장 및 인증).
* **디자인:** 최신 모바일 트렌드를 반영한 고품질 UI/UX.
