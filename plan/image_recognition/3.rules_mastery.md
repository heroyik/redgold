# 🇨🇳 Mastery(마스터) JSON 생성 규칙 (AI Sidekick)

## 🎯 목적
사용자가 제공하는 중국어 학습 자료에서 핵심 문장을 선별하여 **'Mastery(Golden Sentences)'** 섹션을 생성한다. 이 데이터는 앱의 '마스터' 탭에서 렌더링되며, **모든 처리 결과는 해당 레슨의 JSON 파일에 저장되어야 한다.**

## 🛠️ 핵심 요구사항
1.  **데이터 저장소 (Storage)**:
    *   모든 Mastery 문장은 **`data/lesson{번호}.json` 파일의 `key_sentences` 배열**에 저장된다.
    *   앱의 로직은 하드코딩 없이 이 JSON 데이터를 실시간으로 읽어 화면을 구성한다.

2.  **선정 기준 (Selection)**:
    *   해당 과의 본문(Texts) 대화나 독백 중에서 가장 활용도가 높은 **'골든 센텐스' 4~5개**를 엄선한다.
    *   단순한 문장보다는 해당 과의 핵심 문법이나 빈출 표현이 포함된 문장을 우선순위로 둔다.

3.  **데이터 구성 요소**:
    *   **Standard Han (교과서 표준)**: 교재에 있는 원문 그대로의 문장.
    *   **Living Language (실전 구어체)**: 원어민이 실제로 사용하는 생생한 표현. `賊`, `給力` 등 트렌디한 어휘 권장.
    *   **Context**: 사용 상황 태그 (예: "칭찬할 때", "거절할 때").

4.  **하이라이트 규칙 (`<b>` 태그)**:
    *   핵심 문법 패턴은 `<b>` 태그로 강조한다.
    *   UI 연동: 표준 표현은 **Imperial Red**, 구어체 표현은 **Gold** 색상으로 자동 렌더링된다.

## ⚙️ 상세 처리 가이드
1.  **본문 분석**: 추출된 본문 내용 중 학습 가치가 높은 문장을 4~5개 선택한다.
2.  **구어체 변환 및 Enrichment**: AI가 독자적으로 자연스러운 구어체 버전과 뉘앙스 설명을 생성한다.
3.  **JSON 파일 업데이트**: 
    *   `data/lesson{번호}.json` 파일을 열어 `key_sentences` 필드를 찾는다.
    *   기존 데이터가 있다면 최신 내용으로 갱신(Update)하거나 추가(Append)하여 정확성을 유지한다.
4.  **최종 검증**: JSON 스키마(키 이름, 데이터 타입)가 `lesson1.json`과 완벽히 일치하는지 확인한다.
